# API Reference

## Authentication

```http
POST /api/v1/auth/register
POST /api/v1/auth/login
```

Both endpoints return:

```json
{
  "id": "user-uuid",
  "username": "kuku",
  "token": "user-uuid"
}
```

All note and chat requests require:

```http
Authorization: Bearer user-uuid
```

## Notes

```http
POST /api/v1/notes
GET /api/v1/notes
GET /api/v1/notes/{id}
PATCH /api/v1/notes/{id}
DELETE /api/v1/notes/{id}
POST /api/v1/notes/{id}/reindex
```

Create and update body:

```json
{
  "title": "Trip notes",
  "content": "Raw pasted note text..."
}
```

Note response:

```json
{
  "id": "note-uuid",
  "title": "Trip notes",
  "content": "Raw pasted note text...",
  "indexStatus": "pending",
  "indexVersion": 1,
  "indexError": null,
  "indexedAt": null,
  "createdAt": "2026-06-16T00:00:00Z",
  "updatedAt": "2026-06-16T00:00:00Z"
}
```

## Chat

```http
POST /api/v1/chat
```

Request:

```json
{
  "question": "What did I decide about the trip?"
}
```

Response:

```json
{
  "answer": "Answer from saved notes.",
  "notEnoughInfo": false,
  "inference": false,
  "citations": [
    {
      "chunkId": "chunk-uuid",
      "noteId": "note-uuid",
      "noteTitle": "Trip notes",
      "chunkIndex": 0
    }
  ]
}
```
